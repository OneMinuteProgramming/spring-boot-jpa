## Prerequisites
- JDK 17+
- Gradle 7.3+

## Build for source
```shell
gradle clean bootJar
```

## Run the application
```shell
java -jar ./build/libs/spring-boot-jpa-0.0.1-SNAPSHOT.jar
```
